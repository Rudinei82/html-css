Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag b. (não semântica)

Nesta frase, tenho termo em destaque usando a tag strong (semântica).

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântico)

Nesta frase, temos uma Ênfase usando a tag em (semântico). Clica na teclas de atalho CTRL+SHIFT+P e ENTER, para selecionar parágrafo e CTRL+SHIFT+P e EM para italico no selecionado.

Textos Marcados

Podemos tamém criar um texto marcado usando a tag MARK. (CTRL+SHIFT+P ENTER MARK ENTER)

Texto grande e Pequeno

Estamos criando um texto grande e, um texto pequeno neste parágrafo.

Texto deletado (del)

podemos marcar um texto como excluido para indicar que, ele deve ser lido mas não considerado

Texto Inserido (ins)

podemos marcar como um texto inserido para dar uma ênfase que , ele foi adicionado depois.

Também, neste caso pode ser usado para sublinhar a tag U (Não semantico).

Texto Sobrescrito (sup)

podemos enserir coisas do tipo x20+3

Textos Subscrito (sub)

para enserir coisas do tipo h2oh
